# Support Procedures

## Support Channels
1. **Email**: support@kisansmart.com (Primary)
2. **Web Form**: `kisansmart.com/contact`
3. **Internal Ticket System**: Jira / Zendesk (Future)

## Escalation Levels

| Severity | Description | Response Time | Responsibility |
|---|---|---|---|
| **P0 (Critical)** | System Down, Data Loss, Security Breach. | **< 1 Hour** | Lead Developer / DevOps |
| **P1 (High)** | Major feature broken (e.g., Prediction fails). | **< 4 Hours** | Backend Developer |
| **P2 (Medium)** | Minor bug, UI issue, Login trouble. | **< 24 Hours** | Support Staff / Frontend |
| **P3 (Low)** | Feature request, Typo, Cosmetic. | **< 1 Week** | Product Manager |

## Standard Responses

### 1. Acknowledgement
"Hi [Name], thanks for reaching out. We have received your request (Ticket #123) and will get back to you within 24 hours."

### 2. Login Issue
"Please try resetting your password using the 'Forgot Password' link. If that fails, let us know and we can manually trigger a reset email for you."

### 3. Prediction Error
"Could you please share the input values you used? Ensure that pH is between 0-14 and nutrient values are positive numbers."
