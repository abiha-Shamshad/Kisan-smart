## 1. Introduction <a name="introduction"></a>

**Kisan Smart** is an AI-powered agricultural assistant designed to help farmers optimize their fertilizer usage. By analyzing soil parameters (Nitrogen, Phosphorus, Potassium, pH, etc.), the system recommends the most suitable fertilizer and dosage for specific crops, maximizing yield and sustaining soil health.

### Key Features
- **AI-Driven Predictions**: Utilizing advanced machine learning models (Random Forest, XGBoost) for 95%+ accuracy.
- **Crop-Specific Advice**: Tailored recommendations for Wheat, Rice, Maize, Cotton, and more.
- **History Tracking**: Automatically saves all predictions for future reference.
- **Export Reports**: Download recommendations as PDF or CSV.

## 2. Getting Started <a name="getting-started"></a>

### Registration & Login <a name="registration--login"></a>
1. **Sign Up**: Visit the homepage and click **"Get Started"** or **"Register"**.
2. **Details**: Enter your username, valid email, and a strong password.
3. **Verification**: Check your email inbox for a verification link to activate your account.
4. **Login**: Use your email and password to access the system.

### Dashboard Overview <a name="dashboard-overview"></a>
Once logged in, the **Dashboard** provides a quick snapshot:
- **Total Predictions**: Number of recommendations generated.
- **Recent Activity**: Your latest soil tests.
- **Quick Links**: Fast access to "New Prediction" and "History".

## 3. Fertilizer Recommendations <a name="fertilizer-recommendations"></a>

### Collecting Soil Data <a name="collecting-soil-data"></a>
To get accurate results, you need recent soil test data. Ensure you have values for:
- **N (Nitrogen)**: Ratio in soil.
- **P (Phosphorus)**: Ratio in soil.
- **K (Potassium)**: Ratio in soil.
- **pH Level**: Acidity/alkalinity (0-14).
- **Moisture**: Soil moisture percentage.
- **Temperature / Humidity**: Local weather conditions (often auto-filled or estimated).

### Getting a Prediction <a name="getting-a-prediction"></a>
1. Navigate to the **"Predict"** page.
2. Select your **Crop Type** from the dropdown menu.
3. Enter the soil parameters carefully.
4. Click **"Get Recommendation"**.

### Understanding Results <a name="understanding-results"></a>
The system will display:
- **Recommended Fertilizer**: The specific type (e.g., Urea, DAP).
- **Quantity**: Amount needed per acre/hectare.
- **Confidence Score**: How certain the AI is about this recommendation (e.g., 98%).
- **Explanation**: A brief reason for the recommendation based on nutrient gaps.

## 4. Managing History <a name="managing-history"></a>
Your **History** page is your digital logbook:
- **Search**: Find past tests by crop name or date.
- **Filter**: View only specific crops or confidence levels.
- **Export**: Click **"Download PDF"** for a printable report or **"Export CSV"** for spreadsheet analysis.
- **Delete**: Remove outdated or test entries.

## 6. Troubleshooting <a name="troubleshooting"></a>

| Issue | Solution |
|-------|----------|
| **"Prediction Error"** | Check if all numeric fields contain valid numbers. Ensure pH is between 0-14. |
| **"Login Failed"** | Verify email and password. Click "Forgot Password" to reset. |
| **"Email Not Received"** | Check your Spam/Junk folder. Wait 5 minutes and retry. |
| **System Slow** | Check your internet connection. Refresh the page. |

For further assistance, contact **support@kisansmart.com**.
